    # Redstone

Cliente Redmine para iOS

    # Compatibilidade

* Redmine 1.3.x
* Planio (verificar)

    # Nota

Para usar o app é preciso se cadastrar em http://projetos.owera.com.br ou em outro site de versão compatível do Redmine, que esteja com REST web service ativado.
